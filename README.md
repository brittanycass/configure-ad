<h1>On-premises Active Directory Deployed in the Cloud (Azure)</h1>
This tutorial outlines the implementation of on-premises Active Directory within Azure Virtual Machines. We will create a Domain Controller and a Client VM, ensure connectivity, and set up Active Directory with user accounts. <br />



<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop
- Active Directory Domain Services
- PowerShell

<h2>Operating Systems Used </h2>

- Windows Server 2022
- Windows 10 (21H2)

<h2>High-Level Deployment and Configuration Steps</h2>

- Step 1 Create the Domain Controller named DC-1
- Step 2 Create the Client VM (Windows 10) named Client-1
- Step 3 Ensure Connectivity between the Client and Domain Controller
- Step 4 Install Active Directory
